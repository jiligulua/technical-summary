array:
比寻常array安全，而且效率并没有因此变差，array的元素可属于任何被指明的类型T，size()的结果总是N。若需要一个有固定元素量的序列，class array<>将带来最佳效能，因为内存被分配在stack中，绝不会被重分配，而且拥有随机访问能力。
初始化：可使用空白初始列，或初始列来初始化array，当有初始列中的元素个数不到N时，剩余值将是使用默认构造函数，若时int，则为0。若元素个数超过N，则表达式是不对的。
不能对array的初始列使用小括号语句。

写代码最重要的是什么：安全！


自动排序造成set和multiset的一个重要限制：不能直接修改元素值，因为这样会打乱原本正确的顺序，通常以红黑树实现。
