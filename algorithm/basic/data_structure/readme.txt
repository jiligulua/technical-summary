哈希表,是一种为查询所设计的数据结构，查找是它的主要功能，至于插入、删除等，都是需要以查找为基础的。
好的散列函数必须具备两个特点：较少的冲突和函数是均匀的（计算出的任一地址的概率是相等的）
常见散列函数，如下：
1.直接定址法：f(v)=a*v+b; 事先知道关键字集合且不大以及连续性好
2.除留余数法：f(v)= v mod p (p为常数，一般取小于或等于表长的最小素数为宜)，不需要事先知道关键字集合，是最常用的构造散列函数的方法
3.平方取中法：关键字呈随机分布状态，事先不了解关键字的全部状况且关键字的位数不太大

处理冲突的方法：
1.开放定址法：Fi=(f(v)+di) mod n i=1,2,..., v(v<= n-1), 增量序列有3种确定方法：线性探测法、二次探测再散列（di=1，-1，2e2,-2e2,3e2,-3e2,...,v2,-v2），伪随机探测(di是一个伪随机数序列)
2.链地址法：同义词放在一个单练表中
3.公共溢出区法：分基本表和溢出表
